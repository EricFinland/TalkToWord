# TalkToWord

**Simple speech to text, for free.**

Talk into any app on your computer. No subscriptions, no word limits, no cloud. Just speak and it types.

## Setup

```
1. Double-click  setup.bat
2. Double-click  start.bat
```

That's it. First launch downloads a small model (~150MB, one time only).

## Usage

- **Hold Ctrl+Win** to record, let go to paste the text
- Or switch to toggle mode: press once to start, again to stop
- Works in any app — browser, Word, Discord, Slack, whatever

Runs silently in the background. Check the system tray (bottom-right, click the ^ caret) to see it's running. The icon changes color: green = ready, red = recording, orange = transcribing.

Right-click the tray icon for settings or to quit.

## Settings

Right-click the tray icon > **Settings** to change:

- Hotkey
- Recording mode (hold / toggle)
- Model size (bigger = more accurate, slower)
- Language
- Run on startup

## Use on Another Computer

Copy the folder, run `setup.bat`, done. Needs Python 3.10+ and a mic.
